# Prompts for Gemini CLI

This file contains a list of prompts given to the Gemini CLI during the development of this project.

- "Read the file. The tasks are written there. I stopped at the point where I was able to start the server." - This was the initial prompt to get the context of the project.
- "Isn't there something else we need to do?" - A prompt to re-evaluate the next steps.
- "When we start the server, everything is fine, so let's move on." - Confirmation to proceed after verifying the server.
- "yes" - A simple confirmation to proceed with the proposed plan.
- "okey keep going" - A prompt to continue with the Lighthouse audit instructions.
- "Of course, I didn't understand what you meant specifically, but it's all good, so we can continue there. Everything is fine with everything else." - A prompt to continue after some confusion.
- "I need this because I probably need to roll back the project a little so that I can commit it." - The prompt that initiated the Git history rewrite.
- "First, I need to connect to the repository, and then I'll look at the commits you made. If you can, please write down which commits you've already made." - A request to review the Git history.
- "Here is the repository. Are you sure you want to write this in the terminal?" - A prompt asking for confirmation on a set of Git commands.
- "I've already uploaded everything we've done to GitHub, but we need to create a commit history." - Clarification on the need to rewrite Git history.
- "Come up with more text for Changelog and change the text in Prompts for Gemini CLI. Also, there's a problem with the repository. For some reason, I can't upload it to another repository again after deleting it because I couldn't delete the history." - The latest prompt requesting content updates and help with a Git issue.
